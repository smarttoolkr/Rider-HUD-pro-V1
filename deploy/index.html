📍 위치정보 수집 · 이용 안내
이 앱은 정밀 위치(GPS) 정보를 사용하여 저장된 장소와의 거리를 계산하고, 근접 시 메모를 음성으로 안내합니다.

• 수집 항목: 위도, 경도 (GPS 정밀 위치)
• 이용 목적: 장소 정렬, 근접 알림, GPS 핀 저장, 날씨 조회, 주소 자동완성
• 수집 시점: 앱 포그라운드 실행 중에만 (백그라운드 수집 없음)
• 저장 방식: 기기 내 IndexedDB + AES-256 암호화
• 외부 전송: Open-Meteo(날씨), 카카오 지도(주소)에 좌표 전송
   (외부 서비스는 좌표를 수집·저장하지 않음)
• 보유 기간: 앱 사용 중 보유, 데이터 삭제 또는 앱 삭제 시 즉시 파기
• 동의 거부 시: 위치 기반 기능 이용 불가
📋 이용약관 요약
• 이 앱은 라이더 본인이 직접 측정·수집한 데이터만 사용합니다
• 모든 데이터는 기기 내에만 저장되며 개발자는 접근할 수 없습니다
• 타인의 개인정보를 무단으로 수집·저장하는 용도로 사용할 수 없습니다
• 메모 내용의 관리 책임은 사용자 본인에게 있습니다
• 운전 중 앱 조작은 도로교통법 위반이며 금지됩니다
